# 本文件已被升级为分级 requirements。完整版本见同级 requirements/ 子目录：
#
#   requirements/
#     ├── all.txt          ← 完整集（DSW 一键安装：pip install -r requirements/all.txt）
#     ├── core.txt         ← 核心 Agent（pydantic + httpx + numpy）
#     ├── pdf.txt          ← PDF 提取（pdfplumber + pymupdf）
#     ├── retrieval.txt    ← FAISS 向量检索
#     ├── train.txt        ← Llama-3.1-8B 微调 + vLLM 推理
#     ├── analysis.txt     ← 数据分析 + 绘图（28 题评测）
#     └── build_index.txt  ← 仅重建 FAISS 索引用（langchain）
#
# 快速选择：
#   1. 仅跑 Agent（不训练）：pip install -r requirements/core.txt -r requirements/pdf.txt -r requirements/retrieval.txt
#   2. 完整 SFT + 评测：    pip install -r requirements/all.txt
#   3. 仅重建 FAISS 索引： pip install -r requirements/build_index.txt
